7.注释：Python还有一些特殊的注释，已完成一些特别的功能，如中文注释，程序的跨平台等。
	中文注释，必须在Python文件的最前面加上
		#-*― coding: UTF-8 -*-
	如果需要使用Python程序运行在windows以外的平台，需要在Python文件的最前面加
		#！/user/bin/python
	利用注释调试程序，合理的注释方法可以检查程序中的错误。另外一种做法是，将正确语句加上注释，单独运行可能有问题的语句，检查语句错误(使用较长代码)。

8.语句的分隔
	Python支持分号，主要通过换行来时识别语句的结束。
	如果一行中书写多条语句，就必须使用分号分隔每个语句，否则Python无法识别语句之间的间隔。
	Python支持多行写一条语句使用"\\"作换行符。
	
9.变量的赋值
	Python中变量不需要申明，变量的赋值操作即是变量声明和定义的过程。
	Python中一次新的赋值，将创建一个新的变量。即使变量名称相同，变量的标识不相同id(x)
	使用元祖可以一次对多个变量赋值。
	Python对变量的回收采用也是垃圾回收机制。
	
10.全局变量
	全局变量是在函数最外面声明
	如果要对该函数进行引用。使用global保留字，global用于引用全局变量。
	# 错误的使用全局变量  
    _a = 1 
    _b = 2 
    def add(): 
		#global _a   #这里声明对全局变量的引用只有这样，下面的_a才是全局变量。
		_a = 3    #没有使用global保留字引用全局变量，这里的_a是局部变量
		return "_a + _b =", _a + _b #这里的_b是全局变量
    def sub():  
		_b = 4 
		return "_a - _b =", _a - _b  #_a全局变量，_b局部变量。
    print add()  
    print sub() 
	
11.Python中没有常量保留字
	可以自定义一个常量类，来实现常量的功能。实现原理，调用类自带的字典__dict__,判断常量是否包含在字典中。如果包含就抛出异常，否则，给新创建的变量赋值。

13.字符串类型

	在输出的字符串中如果含有单引号，需要使用双引号表示字符串。如果字符串里面含有双引号，需要使用单引号表示字符串。

	结构化程序设计主要方法，自顶向下、逐步细化。把需要解决的问题分成若干个任务来完成，再对每个任务进行设计，逐步细化。
	Python的控制台输入函数为input()，输入为字符串必须加引号
	和raw_input()捕获用户的原始输入，用户可以随意输入。使用字符串存储，需要相应的转化
